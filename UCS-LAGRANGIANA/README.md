# UCS-Lagrangiana

Este módulo apresenta um **modelo cosmológico conceitual** baseado em um campo escalar dinâmico Φ.

## Objetivo

Explorar como uma formulação lagrangiana simples pode gerar:
- dinâmica cosmológica
- equação de estado variável
- previsões para H(z)

## Componentes

- `lagrangiana.md`  
  Formulação da ação e do potencial Φ⁴

- `equacoes.md`  
  Derivação das equações de campo e relações cosmológicas

- `nota-cetica.md`  
  Discussão crítica de limitações, validade e escopo

## Postura Científica

Este módulo:
- não propõe nova física fundamental
- não refuta ΛCDM
- assume caráter didático e exploratório
