# Validação Observacional

Este diretório contém testes observacionais exploratórios
aplicados a modelos cosmológicos efetivos desenvolvidos
no eixo técnico da Ponte Zafira.

Objetivo:
- Comparar modelos de forma estatisticamente honesta
- Documentar resultados positivos, neutros ou negativos
- Preservar aprendizado metodológico

⚠️ Aviso:
Resultados aqui não representam validação de física fundamental.
São análises condicionais a datasets específicos.
