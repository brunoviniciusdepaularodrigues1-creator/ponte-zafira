# Comparação Estatística: UCS vs ΛCDM

Este documento serve como o "paper técnico interno" para a comparação entre o modelo UCS (parametrização CPL) e o modelo padrão ΛCDM.

## Metodologia
A análise utiliza uma abordagem de minimização de $\chi^2$ para os datasets de Supernovas (Pantheon+) e Cronômetros Cósmicos.

## Resultados Resumidos
Os resultados indicam uma preferência estatística para o modelo UCS no regime de baixo redshift ($z < 2$) para os dados analisados, conforme detalhado nos arquivos de métricas.

## Conclusão Exploratória
Embora os resultados sejam estatisticamente favoráveis ao UCS neste contexto específico, eles permanecem sob o status de "Exploratório", aguardando testes com dados de BAO e CMB para uma avaliação de consistência global.
