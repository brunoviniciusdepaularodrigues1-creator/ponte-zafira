# Dados Cosmológicos H(z)

Este diretório contém os dados utilizados no eixo técnico-exploratório do projeto Ponte Zafira.

## Origem
- Dados simulados e inspirados em compilações públicas de H(z)
- Referência conceitual: medições tipo DESI (DR2)
- Uso educacional e exploratório

## Conteúdo
- H(z) observado
- Incertezas experimentais
- Comparação entre ΛCDM e modelo UCS-Lagrangiana
- Cálculo de χ² ponto a ponto

## Aviso
Este conjunto **não representa um dataset oficial** e **não faz afirmações físicas fundamentais**.
