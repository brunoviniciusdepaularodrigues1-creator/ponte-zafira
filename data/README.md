# Dados Cosmológicos — Ponte Zafira

Este diretório contém **dados cosmológicos utilizados para fins educacionais e exploratórios**.

## Conteúdo

- Tabelas de comparação entre:
  - Dados observacionais H(z)
  - Modelo ΛCDM
  - Modelo UCS-Lagrangiana

## Origem dos Dados

Os valores observacionais são **inspirados em medições reais** (ex.: DESI), mas **não constituem um repositório oficial de dados experimentais**.

## Finalidade

- Avaliar comportamento qualitativo dos modelos
- Comparar tendências
- Exercitar análise estatística básica (χ²)

## Aviso Importante

Este material:
- não deve ser usado para inferência cosmológica real
- não substitui bases de dados oficiais
