# Dados Cosmológicos H(z)

Este diretório contém os dados utilizados no eixo técnico-exploratório do projeto Ponte Zafira.

## Conteúdo
- Valores de H(z) observacionais inspirados em compilações públicas
- Incertezas experimentais associadas
- Comparação entre:
  - Modelo de referência ΛCDM
  - Modelo exploratório UCS-Lagrangiana

## Observações Importantes
- Os dados NÃO correspondem a um dataset oficial do DESI
- Uso estritamente educacional, conceitual e exploratório
- Nenhuma reivindicação física fundamental é feita
