# LIMITS — Ponte Zafira

## 1. Escopo

Este projeto define uma arquitetura conceitual e estrutural.
Não é uma teoria física comprovada nem um modelo científico validado.

---

## 2. PCU (Camada de Coerência)

A PCU é normativa e abstrata.

Ela:
- define relações
- orienta estrutura

Ela NÃO:
- executa
- simula
- implementa diretamente

---

## 3. Separação de Camadas

O sistema é dividido em:

- Camada Conceitual
- Camada de Coerência (PCU)
- Camada Técnico-Exploratória

Nenhuma camada deve invadir a outra.

---

## 4. Limite de Interpretação

Este projeto não afirma:

- explicar a consciência
- descrever o universo físico
- substituir modelos científicos estabelecidos

---

## 5. Uso Correto

Este sistema deve ser utilizado como:

- estrutura de organização
- base para experimentação
- ferramenta de coerência

Não como verdade absoluta.

---

## 6. Regra de Ouro

Se algo não pode ser testado ou implementado,
permanece no domínio conceitual.

Não deve ser promovido como resultado técnico.
